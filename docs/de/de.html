Deutsche Lokalisierung für dotProject installieren

  1. Man hole sich die deutschen Lokalisierungsdateien german*.tar.gz von http://sourceforge.net/projects/dotmods/.
  2. Man entpacke den Inhalt des gzipten Archivs in das dotProject-Stammverzeichnis (das Archiv enthält die dotProject-Verzeichnissubstruktur). Folgende Verzeichnisse werden hinzugefügt bzw. bei einer Aktualisierung (teilweise) überschrieben: root_dir/locales/de, root_dir/modules/help/de, root_dir/db/, wobei root_dir das dotProject-Stammverzeichnis repräsentiert. Unter den Windows-Umgebungen ersetze man "/" durch "\".
  3. Will man für alle Benutzer die deutsche Lokalisierung als Standard setzen, melde man sich mit Schreib- und Leserechten für die Systemverwaltung und für die Benutzerverwaltung an (admin). Man wechsle z.B. durch Klick auf System Admin bzw. Systemverwalter in das Systemverwaltungsmodul und wähle dort Default User Preferences bzw. Standardeinstellungen für Benutzer. Dort stelle man Locale bzw. Orts- und Sprachanpassung auf de bzw. Deutsch.
    Alternative zu dem gerade beschriebenen Vorgehen: Man ändere in root_dir/includes/config.php die Konfigurationsvariable zu $dPconfig['host_locale'] = "de";, um Deutsch als Standardsprache einzustellen. Dann wird bereits die Anmeldungsseite auf deutsch angezeigt.
  4. Will man als Administrator für einen einzelnen Benutzer die deutsche Lokalisierung als Standard setzen, melde man sich mit Schreib- und Leserechten für die Systemverwaltung und für die Benutzerverwaltung an (admin). Man wechsle z.B. durch Klick auf User Admin bzw. Benutzerverwalter in das Benutzerverwaltungsmodul und wähle dort den gewünschten Benutzer aus der Tabelle. Auf der Detailsseite für diesen Benutzer wähle man Edit User Preferences bzw. Einstellungen bearbeiten aus den Brotkrümeln. Dort stelle man Locale bzw. Orts- und Sprachanpassung auf de bzw. Deutsch.
In dem Abschnitt Aktualisierung der Dokumentation findet man weitere Erklärungen und Hilfestellungen zu oben angeführten Verfahren. Ausserdem findet sich eine Reihe hilfreicher Informationen im dotProject-Hilfesystem.
 
Das deutsche Übersetzungsteam freut sich über Anregungen, Kritik und weitere Teilnahme zur Verbesserung der Übersetzung. Man besuche dazu http://sourceforge.net/projects/dotmods/.
